# SSPG
## 개요 ##
SSPG (Site specific password generator.)
<code>Vue.js</code>와 <code>Scalar</code> 을 사용한 사이트별 비밀번호 생성기.

메인 비밀번호를 이용해 각 사이트에 맞는 비밀번호를 규칙에 맞게 생성해준다.


## 사용 언어 및 프레임워크 ##
- vue.js(ui) + scalar(api)
- typescript
